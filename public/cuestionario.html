3.¿Qué es el relleno (padding) en el Box Model de HTML?

a. La propiedad box-sizing en CSS se utiliza para controlar cómo se calcula el tamaño total de un elemento en el Box Model de HTML. Por defecto, el tamaño total de un elemento incluye el ancho y alto del contenido,
b. El relleno es el espacio entre el contenido de un elemento y su borde. El relleno se utiliza para agregar espacio adicional entre el contenido de un elemento y su borde, y puede ser ajustado usando la propiedad padding en CSS.
c. El tamaño total de un elemento en el Box Model de HTML se calcula sumando el ancho y alto del contenido, el relleno, el borde y el margen.

4.¿Qué es el margen (margin) en el Box Model de HTML?

a.Se puede modificar el Box Model en HTML utilizando CSS. Las propiedades de CSS como padding, border y margin pueden ser utilizadas para ajustar el tamaño y la apariencia de un elemento
b.El Box Model es una forma en que HTML representa visualmente los elementos en una página web. Se refiere a cómo se compone cada elemento en términos de contenido, relleno, borde y margen
c.El margen es el espacio entre un elemento y otros elementos de la página en el Box Model de HTML. El margen se utiliza para separar visualmente los elementos entre sí y puede ser ajustado utilizando la propiedad margin en CSS
